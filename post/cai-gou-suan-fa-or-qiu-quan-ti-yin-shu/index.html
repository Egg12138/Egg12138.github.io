菜狗算法|| 求全体因数

2021-03-14

a % b == 0
如何能比较好地求出呢？这是我目前遇到过最简单的一道。。。

暴力枚举

暴力枚举肯定可以。
而且只是从1到n遍历一下，也只是O(n)

判断因数的方程组完整解析可以得到O(n\sqrt{n}n​)

a % b = 0 ⇒\Rightarrow⇒ b 和 c = a/b 都是a的因数
一般情况b c有大小之分，
于是我们可以试一下把数字从中间分隔开：
1, 2, 3, ..., n\sqrt{n}n​,...n，当n > 5时，我们知道这个式子的左半段长度必然小于n/2
所以我们只需要在左半段遍历，两个大于n\sqrt{n}n​的数不可能同为其因子。
我们引入一个计算数组内元素长度的办法：
int length=sizeof(a)/sizeof(int); 这里的int是我们的整型数组的元素的类型大小。
但是这样只能判定一个数组有多少格。不能判定我们塞进去的一小段元素的数量.

于是我用这种方法：
[ | | | | '\0' |]我们只要判断结束元素'\0'的位置就可以了。
lenth <-- for (idx=0;idx < (sizeof array)/(sizeof type);idx++>){if *(a+i)=='\0', return i;}
