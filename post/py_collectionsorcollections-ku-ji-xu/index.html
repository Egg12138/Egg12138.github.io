collections.namedtuple()

namedtuple(typename, field_names, *, rename=False, defaults=None, module=None)

  Returns a new subclass of tuple with named fields.

方便构建只有属性没有方法的对象， 比如数据库条目

注意到：

对象定义 Point = namedtuple('Point', ['x', 'y'])
之后直接用赋值方式构建实例
>>> a_p = p(11,2)
>>> type(a_p)
<class '__main__.Point'>这是Point类对象，

加和

具体的加和方法是直接拼接。想要拓展则应该用在其他类中自己定义。

>>> b_p
Point(x=1, y=8)
>>> a_p + b_p
(11, 2, 1, 8)
