Py_collections||Arrow日期时间库

2021-03-09

这个包相当简单，我直接把我的即时交互的结果放上来

arrow.now()

>>> arrow.now('Hongkong')
<Arrow [2021-03-09T17:55:12.578693+08:00]>

arrow.utcnow()

>>> arrow.utcnow().ctime()
'Tue Mar  9 09:50:18 2021'
>>> arrow.utcnow().date()
datetime.date(2021, 3, 9)
>>> arrow.utcnow().dst()
datetime.timedelta(0)

它可以被cloned：
>>> arw = arrow.utcnow()
>>> arw
<Arrow [2021-03-09T09:57:25.147040+00:00]>
>>> arw.clone()
<Arrow [2021-03-09T09:57:25.147040+00:00]>这里arw.clone可以用receive接收

arrow.Arrow() 规范化时间表达

>>> arrow.Arrow(2020, 1, 7, 09,45,22)
File "<stdin>", line 1
    arrow.Arrow(2020, 1, 7, 09,45,22)

这玩意儿会报错，它不支持0开头。。。
>>> arrow.Arrow(2020, 1, 7, 9,45,22)
<Arrow [2020-01-07T09:45:22+00:00]>
